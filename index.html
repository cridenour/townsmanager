Towns Manager

Towns Manager is a simple companion tool for the wonderful game Towns. As of v0.1, Towns Manager has the following features:

  • Manage, create and delete "missions" (save games)
  • Find Towns Folder (Checking common OSX, Windows and Linux directories)
  • Launch Towns

Future

As this is just a first release, the following features are planned for the near future. Likely in this order.

  • Backup / Restore - Automatic backups on any file change, UI for restoration.
  • Menu Editor - Reorder, rearrange menus to better fit your flow / move mod-added items
  • Mod Manager - Add support for original MM and MultiMod packages.

I aim for this to be the easiest to use of any companion tools out there for Towns users to install and manage mods and missions and aim for full cross-platform compatibility.
